# Design System (Nutritionist Website)

## Brand Tone
- Clinical + Friendly + Authoritative
- Must feel trustworthy, calm, premium

## Layout
- Max width: 1100–1200px
- 8px spacing system
- Large vertical spacing (airy layout)
- Section-based structure

## Typography
- Font: Inter or similar
- Base: 16px
- Clear hierarchy (H1–H4)
- High readability (important for blog + SEO)

## Colors
- Primary: soft green (health association)
- Base: white / light gray
- Accent: muted warm tone (CTA only)

## Components
- Cards (soft shadow, rounded 8–12px)
- Buttons (clear primary CTA)
- Navbar (simple, sticky)
- Forms (clean, minimal)

## UX Rules
- No clutter
- Clear section separation
- Strong CTA visibility
- Scannable content blocks

## Blog UX
- Highly readable layout
- Proper heading spacing
- Good line length
- No distractions

## Animation (GSAP)
- Use minimal animations only
- fade + translateY (10–20px)
- duration: 0.4–0.8s
- stagger allowed (0.1s)
- no complex timelines
