# Product Requirements — Nutrigetic Website Overhaul

## 1. Goal

Overhaul the existing website to:
- Improve UI/UX quality
- Preserve and improve SEO rankings
- Increase consultation conversions
- Introduce scalable content system (blog + CMS)

---

## 2. Core Objectives

- Fix SEO issues (missing H1, meta, structure)
- Improve visual hierarchy and trust
- Create conversion-focused layout
- Maintain existing URLs where possible

---

## 3. Target Users

- Professionals (25–45)
- Health-conscious beginners
- SEO-driven visitors (blog traffic)

---

## 4. Core Pages

- Home (conversion-focused)
- About
- Services
- Contact
- Blog (listing + detail)

---

## 5. Features

### Primary
- Consultation booking CTA

### Secondary
- Blog system (SEO support)

### CMS
- Blog creation/editing
- SEO fields (meta title, description, slug)

---

## 6. SEO Requirements

### MUST FIX
- Add H1 to all pages
- Add meta descriptions
- Fix heading hierarchy
- Standardize HTTPS links

### STRUCTURE
- Preserve URLs where possible
- Replace /uncategorized/ with meaningful slugs (with redirects)

### CONTENT
- Improve keyword targeting
- Optimize homepage H1

---

## 7. UX Requirements

- Clean, minimal layout
- Clear CTA placement
- Strong trust signals
- Readable blog layout

---

## 8. Performance

- Fast load (<2s)
- optimized images
- minimal JS (GSAP only where needed)

---

## 9. Success Metrics

- No SEO ranking drop
- Improved page engagement
- Increased consultation conversions
